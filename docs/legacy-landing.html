⚽ Sandbox League Simulator

Football Simulator (HTML)

A god-mode fantasy football league simulator. You don't manage a team — you create the whole pyramid: levels, divisions, teams and rules. Then you simulate seasons and watch an emergent story unfold, tuning weather, attendance, corruption, coaches, form and randomness along the way.

Read the Game Design Document View on GitHub

🏗️ Creator-first

Design league systems with levels, divisions, phases and thresholds — go default or go wild.

🎲 Dice-driven sim

Two-half match engine using an Attack-vs-Defence goal table, coach rolls and modifiers.

🌦️ Systemic flavour

Weather, attendance, derbies, motivation, corruption and crowd trouble add drama.

🔗 Shareable

Whole league systems are user-generated and shareable — recreate real leagues, licence-free.
